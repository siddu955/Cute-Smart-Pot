I made a smart pot with oled and set of sensors which can react to my touch and it has a bunch of emotions.

I usd arduino nano as brain of this project and other components as follows:

1. OLED display (0.96 inch).
2. Capacitive touch sensor.
3. Soil moisture sensor.
4. 10k potentiometers X3.
5. RGB LED strip.
6. Building materials.

Things the pot can do:

It has a bunch of emotions like:

1. Happy.
2. Sad.
3. Dizzy.
4. Angry.
5. Exhausted.
6. Sleepy.
7. Butterfly chasing>
8. Curious.

these emotions changes randomly throught the day .

along with these emotions there are 2 more effects like:

1. Crying when soil is dry .
2. joyfull when soil is wet.
